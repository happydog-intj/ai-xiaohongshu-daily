ai-xiaohongshu-daily — Architecture

三大自动化日报流水线 · GitHub Actions + LLM + Playwright + 飞书推送

三大工作流

  • • 📱 daily.yml — AI 日报 · 08:00 CST
  • • 🔥 trending.yml — GitHub Trending · 08:00 & 21:00
  • • 📰 news_digest.yml — HN & a16z · 09:00 CST ⭐
  • • 全部 GitHub Actions 定时触发，无需服务器

核心脚本

  • • generate_post.py — AI 日报生成
  • • trending_issue.py — Trending 追踪
  • • fetch_news.py — HN + a16z 抓取 ⭐
  • • news_digest.py — 日报主流程 ⭐
  • • card_generator.py — 共享卡片生成

飞书推送方式

  • • App API — 私信图片卡片 + 文字链接
  • • Webhook — 纯文字降级备用
  • • 卡片类型：feishu_card / news_card
  • • 月度 API 成本估算 < ¥30
